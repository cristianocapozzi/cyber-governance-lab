# *Cyber Governance Lab* 
## 📌 Overview
A personal laboratory for cybersecurity governance, regulatory compliance, and ICT risk management. This repository collects practical tools, templates, and study resources for implementing digital resilience frameworks in real-world contexts.

## 📂 Structure
The repository is organized into thematic areas:
- **DORA** → Incident reporting models, ICT third-party registers, and compliance templates
- **NIS2** → Compliance mappings and governance resources
- **ISO 27001** → Risk management and control frameworks
- **Templates** → Reusable operational artifacts and working documents
- **References** → Regulatory and methodological resources

## Objectives
This project aims to translate regulatory and theoretical knowledge into practical, operational tools that can support ICT governance and compliance activities in real organizational contexts.

## Disclaimer
This repository is a personal initiative created for study, research, and professional development purposes. It does not represent any official organization.
